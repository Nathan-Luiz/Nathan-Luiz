## Projeto em destaque
### Snow AI

Uma assistente de IA local desenvolvida em Python utilizando Ollama.
Acredito que IA local não precisa ser exclusiva de computadores high-end. Por isso desenvolvo a Snow AI buscando que ela funcione até em hardwares mais modestos.

### Objetivos do projeto

- funcionar em computadores modestos;
- executar modelos localmente;
- ser modular;
- preservar a privacidade do usuário.

Repositório: [Snow-AI](https://github.com/Nathan-Luiz/Snow-AI)
